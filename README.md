# CSS Mini Project: Rothko Painting

## Description

A simple project to practice CSS Box Model. This project is a simple webpage with a Rothko painting.

## Features

**CSS**

-   Box Model
-   Padding
-   Margin
-   Border
-   Background Color
-   Rotate function
